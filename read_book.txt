HTTP (Hyper Text Transfer Protocol) 는 TCP/IP를 기반으로 동작하고 
HTTPS (Hyper Text Transfer Protocol Secure) 는 UDP를 기반으로 동작한다는 것을 처음 알았다.
HTTPS는 HTTP에 데이터 암호화가 추가된 프로토콜이라고 할 수 있을 것 같다. 
SSL / TLS은 웹사이트와 브라우저 사이에 전송되는 데이터를 암호화 하는 표준 기술이다. 
여담으로 Google은 웹 전반의 보안을 개선하기 위해 모든 사이트에 HTTPS를 요구했습니다.
그리고 SSL 보안 사이트에 대해 SEO를 개선 시켜준다고 약속 했습니다.
